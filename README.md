# acqyre
Tool to fetch apps on Windows.

## Getting started
Run the following from a PowerShell window:

```powershell
&([scriptblock]::Create((curl https://raw.githubusercontent.com/acqyre/acqyre/master/Boot.ps1 | select -exp Content)))
```
